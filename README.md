# llm_report_project
llm 리포트 생성 프로젝트

env.example 양식 그대로 api 키만 대입

data -> 전처리 데이터
src -> 기능 및 단계 별 코드
pipeline -> 파이프라인 (전처리 라인 / 입력쿼리및출력 라인)
notebook -> 실험/테스트용 주피터 노트북 (ex.01_crawling_test.ipynb)
tests -> 유닛 테스트 (ex.test_chunker.py)
main -> 프로그램 진입점

! 생성자 init은 해당 역할 맡은 분이 직접 만드셔야합니다 (미리 안만들었어여)
